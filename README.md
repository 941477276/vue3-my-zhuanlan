# bs-vue
`bs-vue`是一个基于`vue3.x`和`Bootstrap UI 4.x`的Vue组件库！它在Bootstrap UI的基础上进行增强改进，使得`bs-vue`
比bootstrap ui更美观，体验友好。

`bs-vue`继承了`Bootstrap`的CSS响应式特性，因此它适用于各种跨端的项目。



## Project setup
```
npm install
```
